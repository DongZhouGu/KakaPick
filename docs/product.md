# 产品说明

## 产品定义

咔咔选（KakaPick）是一款面向专业摄影师和摄影爱好者的本地选片工具。它自动整理连续拍摄和相似照片，让用户更快比较清晰度、表情与瞬间差异，再将评分安全交给 Lightroom，或直接复制入选照片。

主口号：**拍得多，也能选得快。**

功能说明：**连拍自动成组，好片更快看清。**

## 用户与核心问题

核心用户包括高频拍摄的专业摄影师，以及使用相机拍摄活动、人像、婚礼、旅行和连拍题材的摄影爱好者。

用户打开咔咔选，不是为了增加一个照片管理工具，而是为了减少拍摄后的选片时间。大量照片通常存在三个连续问题：照片数量多、相似照片差异小、反复放大与切换造成判断疲劳。产品首先用“更快完成”吸引用户，再用可靠比较、本地处理和安全导出建立信任。

## 核心价值

- **自动整理**：根据拍摄时间、相机信息与画面相似度组织照片组。
- **快速判断**：并排比较、高清查看、连续键盘评分和自动前进。
- **安心本地**：不上传照片，不修改专有 RAW 文件，不直接写 Lightroom 目录库。
- **顺畅交付**：评分可写入 Lightroom 兼容元数据，也可复制入选源文件。

## 主要工作流

1. 用户从首页选择照片文件夹，或打开最近相册。
2. 应用扫描支持的 RAW、JPEG 和 XMP，读取元数据并建立 RAW+JPEG 配对。
3. 应用将连续拍摄和相似照片自动分组，无法确认的信息以警告呈现。
4. 用户在沉浸式工作台中并排比较，使用键盘或指针评分、保留、淘汰和查看高清细节。
5. 评分持续保存；再次打开同一目录时恢复已有会话。
6. 用户完成选片后，将评分写入 Lightroom 兼容元数据，或复制达到星级阈值的照片。

## 功能范围

当前产品包含：

- JPEG/JPG 与 ARW、CR2、CR3、NEF、RAF、RW2、ORF、DNG 扫描；
- RAW+JPEG 配对和 RAW 内嵌预览提取；
- 自动分组、分组灵敏度、手动拆分与合并；
- 最近相册、会话恢复与高清预览；
- 0–5 星评分、保留、淘汰、筛选和撤销；
- 基础清晰度、过曝与欠曝辅助标记；
- Lightroom XMP/嵌入式评分写入与回滚；
- 入选照片及附属文件的安全复制。

## 明确非目标

- 不上传照片，不提供账号、云同步或多人协作。
- 不删除原片，不自动替用户做最终审美决定。
- 不承担调色、裁剪、预设、修图或 RAW 显影。
- 不直接修改 Lightroom catalog 数据库。
- 不把“AI”包装成确定无误的选片结论。

## 产品原则

1. 照片优先：工作台界面必须让位于照片判断。
2. 速度可感知：扫描、保存和导出都要反馈真实进度。
3. 操作可恢复：评分持续保存，危险写入提供预览、确认和回滚。
4. 本地且透明：用户清楚知道文件在哪里、哪些内容会被写入。
5. 专业但不沉重：语言轻松直接，能力说明准确克制。

## 成功标准

- 用户首次进入即可理解产品用于快速完成大量照片筛选。
- 用户能从真实文件夹完成扫描、分组、比较、评分和至少一种导出。
- 关闭并重新打开同一相册后，评分和分组调整可以恢复。
- 任何导出错误都不会静默覆盖或损坏原始文件。
- 高频操作无需离开键盘，所有关键能力也能用指针完成。
